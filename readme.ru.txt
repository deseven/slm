Sins Localization Merger
by deseven, 2011

=== Описание ===
Это утилита для совмещения двух разных файлов локализации.
К примеру у вас есть русская версия SoaSE и вы хотите поиграть в нее с каким-нибудь модом. Вы ставите мод и опа - ваша русская версия превращается в английскую.
Данная утилита берет все доступные строки из файла локализации и добавляет их в мод. Таким образом получается полу-локализованный мод. Это также очень полезно для выполнения переводов, т.к. все нелокализованные строки окажутся в самом низу файла локализации.  

=== Совместимость ===
Утилита работает на Windows 2000/XP/Vista/7 и совместима со всеми версиями SoaSE, по крайне мере пока формат файла локализации остается таким же.
	
=== Использование ===
Просто запустите slm.exe и выберите основной файл (ваш файл локализации, расположен в <sins directory>\String\).
Потом выберите вторичный файл (файл локализации мода, расположен в <mod directory>\String\).
Наконец выберите итоговый файл (вы можете сразу выбрать файл локализации мода чтобы провести слияние на лету) и нажмите GO!

Процесс слияния может занять некоторое время на старых компьютерах, просто подождите пока утилита не напишет о завершении всех задач.

=== Возможные проблемы ===
Я не делал никаких проверок для различных вариантов форматирования файла локализации. На текущий момент файл локализации должен иметь дефолтный вид:

TXT
NumStrings <количество строк>
StringInfo
	ID "someID1"
	Value "someValue1"
StringInfo
	ID "someID2"
	Value "someValue2"
...
